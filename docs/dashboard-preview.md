# Dashboard preview

The image **dashboard-preview.png** in this folder is a mockup of the dashboard layout.

**To view the image:** open `docs/dashboard-preview.png` in Finder, your editor, or any image viewer.

**Dashboard layout:** `tg dashboard` (default) shows two stacked tables: **Active Projects** (top) and **Active tasks and upcoming** (bottom). Row counts are capped from terminal height so the screen does not scroll. Run `pnpm build` then `tg dashboard`.
